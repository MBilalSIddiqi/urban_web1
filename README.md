# URBN_ZN — The Drop

A static, multi-page demo website for a fictional underground streetwear / "zine" collective — *"Urban Zine Collective."* Bold, gritty, brutalist anti-corporate tone. Built with plain HTML, CSS, and vanilla JavaScript (no framework or build step).

**Live site:** https://mbilalsiddiqi.github.io/urban_web1/

## Pages
- `index.html` — THE DROP (landing)
- `culture.html` — THE UNDERGROUND (manifesto/about)
- `lookbook.html` — THE VISION (lookbook)

## Tech
- HTML5, CSS3 (custom properties, thick borders, hard offset shadows, glitch transitions), vanilla JavaScript
- Palette: black `#000000`, white, acid neon-lime `#BBFF00`
- Google Fonts: Archivo Black + JetBrains Mono
- Shared `styles.css` and `script.js`

## Run locally
No build step required — open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000   # visit http://localhost:8000
```
